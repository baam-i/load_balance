# Proyecto de Balanceo de Carga
**Pasos para ejecutar**
1. Descargar o clonar proyecto.

   1.1 Clonar el proyecto ```git clone https://github.com/baam-i/load_balance.git```
3. Recomendamos generar un entorno virtual de python 3.11 o mayor para ejecutar el proyecto
   
   2.1 Crear entorno: ```python -m venv entorno```
   
   2.2 Activar entorno: ```source entorno/Scripts/activate```
   
   2.3 Desactivar entorno (Cuando se haya terminado la ejecucion del proyecto) ```deactivate```
   
4. Instalar las dependiencias del proyecto desde consola ```pip install -r requirements.txt```.
5. Tras la instalación de librerías, ya podemos ejecutar. Desde consola sería con ```python main.py```.
